Comparación OSI vs TCP/IP

Modelo OSI

El modelo OSI (Open Systems Interconnection) es un modelo conceptual que describe cómo las aplicaciones de red se comunican entre sí en un entorno de red. Está compuesto por siete capas:

  • Capa 7: Aplicación
  • Capa 6: Presentación
  • Capa 5: Sesión
  • Capa 4: Transporte
  • Capa 3: Red
  • Capa 2: Enlace de datos
  • Capa 1: Física

Modelo TCP/IP

El modelo TCP/IP (Transmission Control Protocol/Internet Protocol) es el conjunto de protocolos utilizado en Internet y en la mayoría de las redes locales. Está compuesto por cuatro capas principales:

  • Capa de Aplicación
  • Capa de Transporte
  • Capa de Internet
  • Capa de Acceso a la Red
